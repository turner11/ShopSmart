1. In order to work with program you need first install the DB on a local SQL Server, the Instance name must be "SQLEXPRESS".
2. In order to install db you can run the BuildDbScript.sql script on the specified instance.
3. Install the program using THE "Deployed_Items\setup.exe" file.
4. For usage, please refer to the Usage document (https://github.com/turner11/ShopSmart/blob/master/Installation_And_usage/UsageSnapShots.docx?raw=true)
5. As default user you can use: user/password: Admin/123 (case sensitive)
